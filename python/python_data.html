# 数据类型

python 作为一门动态语言，最具特色的就是其变量可以随时变更为任意一种数据类型。

在使用过程中，一般都是基于 puthon 原生的基本数据类型 int str list dict 等。

但是也有的库会因为特殊的需求开发出了如 MultiIndex，datetime，DataFrame 等新的数据类型和数据容器。
