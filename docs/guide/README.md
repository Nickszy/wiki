# 我的学习笔记

希望能够通过这个项目将自己学习到的东西整理成体系的学习资料供大家参考，成为大家学习互相印证的参考。

学习笔记大概分为 `数据科学`、`区块链`、`应用`这三个部分。

## 具体内容

`大数据` ：包含数据库，数据挖掘，数据分析，数据可视化这几个模块。

`前端`：借助`Django`快速部署项目，`vue`可以用来是项目变得好看，至于`flutter`则是多端部署的利器。

`区块链`：根据区块链不同的层分别进行介绍，随后开发一个`Dapp`。
