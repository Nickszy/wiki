# Python

我觉得python是万能的，学好python走遍天下都不怕。

## python 攻城獅

大概分为三个不同的岗位：

1. 后端开发
  Linux+mysql+web框架，tensorflow有加分

2. 前端/全栈
  H5+前端框架+python框架，数据库加分

3. 算法、大数据、爬虫
  爬虫+数据库+算法，数学+DeepLearning经验加分

## 如何学习

linux：[鸟哥的私房菜](http://linux.vbird.org/)、进阶-> apue《环境高级编程》
