## git 的功能点

- 文件记录追踪
- 分支
- tags
- github

## 文件状态

总共有 4 种状态。

1. untracked  // 为进行追踪的文件
2. modified  // 修改过，未 commit
3. unmodified  // commit 后无修改
4. stage  // 暂存
